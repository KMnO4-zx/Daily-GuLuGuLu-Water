# Daily GuLuGuLu Water

- 单片机学习仓库，希望我能从LED开始，直到年底完成**Daily GuLuGuLu Water**项目
- 一个利用esp32+蓝牙+wifi+LCD+压力传感器的检测每日喝水次数与喝水量的智能杯垫项目。
- 我真的很需要这个杯垫！

## 技术路线

- 利用压力传感器记录杯子放到杯垫上时的数值，经过一次喝水后，杯子重新放回杯垫，此时记录差值即为喝水量。
- 可以统计喝水的次数和每天的总的喝水量。
- 目前打算利用蓝牙或wifi实现与PC端和手机端的通信。（实时查看饮水量）
